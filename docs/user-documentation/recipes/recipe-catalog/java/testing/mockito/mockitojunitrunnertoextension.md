# Replace JUnit 4 MockitoJUnitRunner with junit-jupiter MockitoExtension

**org.openrewrite.java.testing.mockito.MockitoJUnitRunnerToExtension**

_Replace JUnit 4 MockitoJUnitRunner annotations with JUnit 5 `@ExtendWith(MockitoExtension.class)` using the appropriate strictness levels (LENIENT, WARN, STRICT_STUBS)._

[GitHub: MockitoJUnitRunnerToExtension.java](https://github.com/openrewrite/rewrite-testing-frameworks/blob/main/src/main/java/org/openrewrite/java/testing/mockito/MockitoJUnitRunnerToExtension.java),

## Example


<Tabs groupId="beforeAfter">
<TabItem value="java" label="java">


###### Before
```java
import org.junit.runner.RunWith;
import org.mockito.junit.MockitoJUnitRunner;

@RunWith(MockitoJUnitRunner.Strict.class)
public class ExternalAPIServiceTest {
}
```

###### After
```java
import org.junit.jupiter.api.extension.ExtendWith;
import org.mockito.junit.jupiter.MockitoExtension;

@ExtendWith(MockitoExtension.class)
public class ExternalAPIServiceTest {
}
```

</TabItem>
